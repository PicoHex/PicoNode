# OpenAI / Multi-Provider Thinking Support Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Migrate thinking/reasoning configuration from hardcoded values to `settings.json`-driven approach, add OpenAI `reasoning_effort` support, and introduce per-model thinking defaults with session persistence.

**Architecture:** Data flows from `settings.json` → `AgentConfig` → `Model` (with `ThinkingEnabled` + `ThinkingLevel` + `ThinkingLevelMap`) → `StreamOptions` → LLM Client `BuildRequestJson`. Level resolution uses `MapResolver` (nearest-match algorithm). Session state persisted via `SessionData` in JSONL `$meta` line.

**Tech Stack:** C# / .NET 10, PicoJetson (source-gen JSON), TUnit, System.IO.Pipelines

## Global Constraints

- All default values from `settings.json` — zero hardcoded thinking defaults in code beyond `ThinkingLevel.Medium` as a single fallback for missing `thinkingLevel`
- TDD: every behavior change has a failing test before implementation
- `Model.ThinkingLevel` non-nullable, explicitly assigned — never relies on `default(ThinkingLevel)`
- Empty `{}` thinking map treated as "not supported" (equivalent to `null`)
- MapResolver called in LLM Client, not AgentLoop
- `settings.example.json` shipped as publish Content, not auto-generated by code

```csharp
// src/PicoNode.AI/LLm/MapResolver.cs
namespace PicoNode.AI;

/// <summary>
/// Resolves a ThinkingLevel to a provider-specific API parameter value.
/// Strategy: exact match → nearest downward (cheaper) → nearest upward → null.
/// </summary>
public static class MapResolver
{
    private static readonly ThinkingLevel[] Ordered =
        [ThinkingLevel.Minimal, ThinkingLevel.Low, ThinkingLevel.Medium, ThinkingLevel.High, ThinkingLevel.XHigh];

    public static string? Resolve(ThinkingLevel requested, Dictionary<string, string>? map)
    {
        if (map is null || map.Count == 0)
            return null;

        // 1. Exact match
        var key = requested.ToString().ToLower();
        if (map.TryGetValue(key, out var value))
            return value;

        // 2. Downward (cheaper)
        for (int i = (int)requested - 1; i >= 0; i--)
        {
            key = Ordered[i].ToString().ToLower();
            if (map.TryGetValue(key, out value))
                return value;
        }

        // 3. Upward
        for (int i = (int)requested + 1; i < Ordered.Length; i++)
        {
            key = Ordered[i].ToString().ToLower();
            if (map.TryGetValue(key, out value))
                return value;
        }

        return null;
    }
}
```

```csharp
// SessionStore.cs — approximate changes:
public static async Task<SessionData> LoadAsync(string path)
{
    if (!File.Exists(path))
        return new SessionData();

    var lines = await File.ReadAllLinesAsync(path);
    var result = new SessionData();

    // Parse $meta line if present
    if (lines.Length > 0 && lines[0].StartsWith("{\"$meta\""))
    {
        var metaJson = lines[0];
        try
        {
            using var doc = JsonDocument.Parse(metaJson);
            if (doc.RootElement.TryGetProperty("$meta", out var meta))
            {
                if (meta.TryGetProperty("thinkingEnabled", out var te))
                    result.ThinkingEnabled = te.GetBoolean();
                if (meta.TryGetProperty("thinkingLevel", out var tl))
                    result.ThinkingLevel = Enum.TryParse<ThinkingLevel>(tl.GetString(), ignoreCase: true, out var l) ? l : ThinkingLevel.Medium;
            }
        }
        catch { /* ignore malformed meta */ }

        lines = lines.Skip(1).ToArray(); // Skip meta line for message parsing
    }

    // Parse message lines (existing logic)
    // ...

    return result;
}

public static async Task SaveAsync(string path, SessionData data)
{
    var sb = new StringBuilder();
    sb.Append("{\"$meta\":{");
    sb.Append($"\"thinkingEnabled\":{data.ThinkingEnabled.ToString().ToLower()},");
    sb.Append($"\"thinkingLevel\":\"{data.ThinkingLevel.ToString().ToLower()}\"");
    sb.AppendLine("}}");

    foreach (var msg in data.Messages)
    {
        sb.AppendLine(PicoJetson.JsonSerializer.Serialize(msg));
    }

    await File.WriteAllTextAsync(path, sb.ToString());
}
```

```csharp
// Rewrite Apply method to handle level arguments
public static string? Apply(Model model, string arg)
{
    if (string.IsNullOrWhiteSpace(arg))
    {
        // Bare /thinking — toggle
        model.ThinkingEnabled = !model.ThinkingEnabled;
        return null;
    }

    if (arg is "on")
    {
        model.ThinkingEnabled = true;
        return null;
    }

    if (arg is "off")
    {
        model.ThinkingEnabled = false;
        return null;
    }

    // Level arguments: enable + set level
    var parsed = AgentConfig.ParseLevel(arg);
    if (parsed is { } level)
    {
        model.ThinkingEnabled = true;
        model.ThinkingLevel = level;
        return null;
    }

    return "Usage: /thinking [on|off|minimal|low|medium|high|xhigh]";
}
```

### Task 8: Update AgentHost — Per-Session Thinking State

**Files:**
- Modify: `src/PicoNode.Agent/Host/AgentHost.cs`

**Design note:** `Model` is shared across all sessions. Thinking state (enabled/level) must be per-session to avoid cross-session contamination. AgentHost holds a `ConcurrentDictionary<string, SessionThinking>` keyed by sessionId.

**Interfaces:**
- New type: `SessionThinking { bool ThinkingEnabled; ThinkingLevel ThinkingLevel; }`
- Consumes: `SessionData.ThinkingEnabled/Level`, `Model.ThinkingEnabled/Level` (config defaults)
- Produces: `AgentHost` with per-session state management

- [ ] **Step 1: Add SessionThinking record and per-session dictionary**

```csharp
// src/PicoNode.Agent/Host/AgentHost.cs
namespace PicoNode.Agent;

public sealed class AgentHost
{
    private readonly AgentLoop _loop;
    private readonly ConcurrentDictionary<string, List<Message>> _sessions = [];
    private readonly ConcurrentDictionary<string, SessionThinking> _thinkingState = [];  // NEW

    public AgentHost(AgentLoop loop)
    {
        _loop = loop;
    }

    /// <summary>Get or set thinking state for a session. Falls back to global defaults.</summary>
    public SessionThinking GetThinkingState(string sessionId, bool defaultEnabled, ThinkingLevel defaultLevel)
    {
        return _thinkingState.GetOrAdd(sessionId, _ => new SessionThinking
        {
            ThinkingEnabled = defaultEnabled,
            ThinkingLevel = defaultLevel,
        });
    }

    public void SetThinkingState(string sessionId, bool enabled, ThinkingLevel level)
    {
        _thinkingState[sessionId] = new SessionThinking { ThinkingEnabled = enabled, ThinkingLevel = level };
    }

    /// <summary>Restore thinking state from persisted SessionData.</summary>
    public void RestoreSession(string sessionId, SessionData data)
    {
        _sessions[sessionId] = data.Messages;
        _thinkingState[sessionId] = new SessionThinking
        {
            ThinkingEnabled = data.ThinkingEnabled,
            ThinkingLevel = data.ThinkingLevel,
        };
    }

    public SessionData GetSessionData(string sessionId)
    {
        var messages = _sessions.GetValueOrDefault(sessionId) ?? [];
        var thinking = _thinkingState.GetValueOrDefault(sessionId)
            ?? new SessionThinking { ThinkingEnabled = true, ThinkingLevel = ThinkingLevel.Medium };
        return new SessionData
        {
            Messages = messages,
            ThinkingEnabled = thinking.ThinkingEnabled,
            ThinkingLevel = thinking.ThinkingLevel,
        };
    }

    // Existing methods... (ProcessMessageAsync, GetSessionMessages)
}

public sealed class SessionThinking
{
    public bool ThinkingEnabled { get; set; }
    public ThinkingLevel ThinkingLevel { get; set; }
}
```

- [ ] **Step 2: Update ProcessMessageAsync to copy thinking state to Model**

```csharp
public async Task<string> ProcessMessageAsync(
    string content, CancellationToken ct,
    string sessionId = "default",
    Func<AssistantMessageEvent, CancellationToken, ValueTask>? onEvent = null)
{
    var messages = _sessions.GetOrAdd(sessionId, _ => []);
    var thinking = _thinkingState.GetValueOrDefault(sessionId)
        ?? new SessionThinking { ThinkingEnabled = true, ThinkingLevel = ThinkingLevel.Medium };
    _loop.UpdateThinking(thinking.ThinkingEnabled, thinking.ThinkingLevel);

    messages.Add(new Message { Role = RoleUser, Content = content, Timestamp = DateTimeOffset.UtcNow.ToUnixTimeMilliseconds() });

    var result = await _loop.RunTurnAsync(messages, ct, onEvent);

    var lastAssistant = result.LastOrDefault(m => m.Role == RoleAssistant);
    return lastAssistant?.ContentBlocks?.Where(cb => cb.Type == BlockTypeText).Select(cb => cb.Text).FirstOrDefault() ?? "";
}
```
